## What is dbt?

dbt (data build tool) enables analytics engineers to transform 
data in their warehouses by writing SQL select statements.
It handles the T in ELT (Extract, Load, Transform).

---

## Key Concepts

### Models
SQL files that define transformations on raw data.

### Sources
Raw data tables in your warehouse that dbt reads from.

### Tests
Assertions about your data that ensure quality and integrity.

### Documentation
Auto-generated docs from your dbt project.
